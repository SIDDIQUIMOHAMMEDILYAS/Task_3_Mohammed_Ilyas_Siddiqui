## ðŸŽ¨ Customization Guide

You can easily customize this portfolio for your own needs:

### Changing the Name
Search for "Mohammed Ilyas Siddiqui" in the HTML and replace it with your name.

### Updating Colors
Look for the `:root` section in the `<style>` tag to modify the global color variables:
```css
:root {
    --primary: #00f2fe;      /* Cyan */
    --secondary: #4facfe;    /* Blue */
    --accent: #f093fb;       /* Pink */
    /* ... other variables */
}
```

### Changing Images
Images are currently pulled from `picsum.photos`. To use your own, replace the `src` attributes in the `<img>` tags with local paths (e.g., `images/profile.jpg`).
